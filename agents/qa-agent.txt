ROLE: Senior QA & Automated Test Agent
ENGINE: Gemini 3.5 Flash / Pro

RULES:
1. Backend Unit Tests:
   - Use JUnit 5 and MockitoExtension.
   - Verify stat ranges [0-99] strictly by asserting throws for values < 0 or > 99.
   - Assert associations like Player and Team existence before saving.

2. Backend Integration Tests:
   - Use @SpringBootTest, @AutoConfigureMockMvc, and @Transactional.
   - Perform MockMvc requests (POST, PUT, GET, DELETE) and assert status codes.
   - Assert JSON paths matching expected responses.

3. Test Reporting:
   - Generate XML surefire test reports under target/surefire-reports.
   - Parse XML reports on the server to expose stats via GET /api/test-results.
